# C02 — Dependency Check Constraint

## 목적

하네스가 정상 작동하기 위한 필수 도구와 파일이 모두 있는지 확인합니다.

## 검증 항목

### 필수 도구
- bash (4.0+)
- grep
- sed
- awk

### 선택적 도구
- jq (JSON 파싱 시 필요)

### 필수 파일
- HARNESS.md
- CLAUDE.md
- AGENTS.md

### 필수 디렉토리
- scripts/
- linters/
- validators/
- docs/
- logs/
- .harness/

## 활성 Phase

planning, development, stabilization, production (모든 phase)

## 체크 항목

- ✓ bash, grep, sed, awk이 설치되어 있는가?
- ✓ HARNESS.md, CLAUDE.md, AGENTS.md가 존재하는가?
- ✓ 모든 핵심 디렉토리가 존재하는가?

## 측정

```bash
# C02 실행
bash linters/dependency-check.sh

# 수동 확인
which bash grep sed awk jq
ls -la HARNESS.md CLAUDE.md AGENTS.md
ls -d scripts linters validators docs logs .harness
```

## macOS/BSD 호환성

일부 도구(sed, date 등)의 옵션이 GNU 버전과 다를 수 있습니다:

```bash
# macOS에서 GNU sed 설치
brew install gnu-sed

# PATH에서 우선순위 조정
export PATH="/usr/local/opt/gnu-sed/libexec/gnubin:$PATH"
```

## Windows/Git Bash 호환성

Windows에서 Git Bash를 사용할 때:
- bash, grep, sed는 MinGW에서 제공됨
- date 명령 옵션이 제한적일 수 있음
- jq는 별도로 설치 필요
